# Habbo-Assets

## How to install

```bash
# install with npm
npm i --save habbo-assets

# install with yarn
yarn add habbo-assets
```

## How to use

```tsx
import as * React from 'react'

import { VipIcon } from 'habbo-assets'

const habboAssets = () => {
    return (
        <>
            <VipIcon /> {/*  you can use img tag properties (style, alt, etc) */}
        </>
    )
}

export default habboAssets
```
